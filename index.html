Hobby Projects

  • Appointment Scheduler

    Finished on: April, 2019

    Java Spring Hibernate FlyingSaucer FullCalendar Thymleaf MVC

    This is so far my biggest project. I created it for my friend who needed a custom platform to schedule appointments between his providers (teachers) and his customers (students). Admin can add providers and available services. Providers can set their weekly working plan, add breaks to their plan and choose which service they offer. Customers can book appointments for their selected service and provider with just a few clicks. Every appointment has 7 possible statuses: scheduled, canceled, finished, confirmed, rejection requested, rejection accepted, invoiced. Normal lifecycle of appointment is: scheduled -> finished -> confirmed -> invoiced. On the 1st day of each month invoices are issued automaticaly for every customer for their all confirmed appointments. For every appointment status change an email notification is sent to provider/customer. More informations about logic (cancelation rules, rejections rules etc) can be found on github readme page.

  • HowLong App

    Finished on: March, 2018

    Java Android Dagger Retrofit2 Butterknife MVP

    This is my first android application created while learning Android SDK. The main purpose of this application is to inform users what is the mean waiting time in a choosen restaurant. It allows user to do two things: add new restaurant which is not yet int the database or add new time report to an existing restuarant. Every time reports expires after 1 hour and is deleted automatically. The mean waiting time for each restuarnat is just an arithmetic mean calculated from active time reports. It communicates with google Places API to retrieve restaurant details and with HowLong App REST Api (described above) to store time reports and restaurasnt details in remote database.

  • Speed measurement with radio waves

    Finished on: June, 2016

    C Arduino Hardware

    For my master's thesis I created a device to measure speed with the use of 10.525 GHz radio waves. The source of radio waves was an HB-100 module, used in automatic doors openers. Calculation and speed presentation was done using a 32-bit microcontroller from the STM32 family. The aim of the thesis was to create a mobile device to measure the speed of a moving object with the measuring range being in between 0.04 and 0.99 m/s. The main tasks involved building the reference movement platform, making an amplifier board to gain IF signals from the HB-100 module and writing a speed calculation algorithm. Measurement results obtained from this device were similar to results obtained from laser Doppler vibrometer. Unfortunately I lost source code.

  • 2-axis Active Camera Stabilizer

    Finished on: February, 2015

    C Arduino Hardware

    This project was made as my engineering project. The purpose was to design and create camera stabilizer for small cameras like GOPRO or mobile phones. This device works by measuring angle of rotation in two axis – X and Y. Then the opposite value of each angle is send to the two DC engines respectively for X-axis and Y-axis. Because of that, camera is leveled all the time and operator moves doesn’t affect it. The measuring chip is MPU-6050 produced by IvenSense. It contains 3- axis accelerometer and 3-axis gyroscope. Angles between sensor-board and gravity vector are calculated by fusion of accelerometer and gyroscope readings. Readings from accelerometer are calculated with trigonometric functions. Readings from gyroscope are in angular velocity. So by integrating them over time we get angle of rotation. All calculations are processed by Arduino and then the opposite value of calculated angle is send as PWM signal to analog servos. Unfortunately I lost source code.
